###解决git无法push的问题
* 添加sendpack.sideband属性并置为false就解决了。

* 全局的：git config --global sendpack.sideband false

* 仓库的：git config --local sendpack.sideband false

* 通过git config --local -l 查看仓库级配置，可以看到有sendpack.sideband这一项并且是置为false的。

* 再次push提交到远程仓库已经ok了。
